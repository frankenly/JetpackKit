# JetpackKit

A description of this package.

# Debug
How to use ?
```swift
    
    // using 
    JPKDebug.print("Subject", msg: "Detail", mode: .LOG)

    // disable - enable
    JPKConfigure.isDebug = false
```
* mode
    * .LOG is white icon
    * .ERROR is red icon
    * .SUCCESS is green icon
