# Debug
How to use ?
```swift
    
    // using 
    FKDebug.print("Subject", msg: "Detail", mode: .LOG)

    // disable - enable
    FKConfigure.isDebug = false
```
* mode
    * .LOG is white icon
    * .ERROR is red icon
    * .SUCCESS is green icon
